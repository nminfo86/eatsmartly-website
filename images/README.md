## 🎨 Image Creation Tips (AnyDesk Style)

1. **Use Clean Screenshots:**
   - Take screenshots of your actual application
   - Remove any test/dummy data
   - Use professional sample data

2. **Add Professional Touch:**
   - Use tools like Figma, Photoshop, or online tools
   - Add subtle shadows for depth
   - Ensure good lighting and clarity

3. **Consistent Style:**
   - All screenshots should have the same style
   - Use the same color scheme (#0b7a61 primary)
   - Maintain consistent padding/margins

4. **Mobile Mockups:**
   - Use mockup generators like:
     - https://mockuphone.com/
     - https://www.screely.com/
     - https://shots.so/

5. **Optimize Images:**
   - Save as PNG for quality
   - Compress to reduce file size (use tinypng.com)
   - Keep under 500KB per image
