**Features:**

- High-quality images and video support

- Multi-language support (Arabic, French, English)

- Smooth and fast ordering process

### 4. **Food Court Detail Section** 🏪
**Capabilities:**
- Unified ordering system for multiple restaurants
- Ordering via smartphones, self-service kiosks, or interactive screens
- Automatic routing to correct kitchen
- Real-time order tracking
- Updates displayed on TV screens or customer devices
- Distributed printing system (each kitchen receives only its orders)
- Centralized management dashboard
- Separate and consolidated reporting

**Ideal For:**
- Shopping malls
- Airports
- University cafeterias
- Event halls

**Key Feature:** 100% offline operation via local network only

### 5. **Printers and Kitchen Tickets Section** 🖨️
**Printer Support:**
- ESC/POS thermal printers (80mm for invoices)
- EZPL/TSPL printers for kitchen tickets (various sizes)
- Smart routing by category or prep station
- USB and network printer support
- Customizable invoice and ticket designs
- Automatic printing on order receipt
- Multiple kitchen printing support

**Benefit:** Each kitchen receives only the items it prepares, improving efficiency and reducing errors

### 6. **Operational Scenarios Section** ✅
**4 Supported Modes:**

#### Scenario 1:
- **Type:** POS with single USB printer
- **Ideal for:** Small cafes and startup restaurants
- **Description:** Simple and effective solution with one directly connected printer

#### Scenario 2:
- **Type:** POS with multiple printers
- **Ideal for:** Medium restaurants with multiple kitchens
- **Description:** Smart distribution of printing by dish type

#### Scenario 3:
- **Type:** Multi-printer POS + QR Menu with customer orders
- **Ideal for:** Modern restaurants
- **Description:** Complete solution with POS + digital menu + smart kitchen printing

#### Scenario 4:
- **Type:** Food Court mode with multi-restaurant system
- **Ideal for:** Large food courts
- **Description:** Centralized management of multiple restaurants with smart order routing
